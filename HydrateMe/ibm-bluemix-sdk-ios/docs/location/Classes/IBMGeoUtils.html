IBMGeoUtils Class Reference

Inherits from NSObject
Declared in IBMGeoUtils.h

Overview

Provides access to utility functions for Geo calculations.

The IBMGeoUtils class is a provisional API. APIs that are marked provisional are evolving and might change or be removed in future releases.

getDistanceFromCoordinate:toArea:

Calculates the distance of the coordinate from the area. Equivalent to calling getDistanceFromCoordinate:toArea:withBufferZoneWidth: with a 0 as the bufferZoneWidth parameter.

+ (double)getDistanceFromCoordinate:(IBMPosition *)coordinate toArea:(id<IBMArea>)area

Parameters

coordinate

The coordinate to measure distance to

area

The area to measure distance from

Return Value

the distance in meters to the area. The distance is positive for coordinates outside the area and negative for coordinates inside the area.

getDistanceFromCoordinate:toCircleWithCenter:radius:withBufferZoneWidth:

Calculates the distance of the coordinate from the circle, taking into account the buffer zone.

+ (double)getDistanceFromCoordinate:(IBMPosition *)coordinate toCircleWithCenter:(IBMPosition *)center radius:(double)radius withBufferZoneWidth:(double)bufferZoneWidth

Parameters

coordinate

The starting coordinate

center

The center used to define the circle

radius

The radius used to define the circle

bufferZoneWidth

The buffer zone width, measured in meters. It enlarges the radius of the circle by this amount. Negative values make the circle smaller.

Return Value

(double) The distance, in meters, to the circle, taking into account the buffer zone. The distance is positive for coordinates outside the circle, and negative for coordinates within the circle. Will return NaN if buffer zone width is negative and the absolute value is greater than the circle’s radius.

getDistanceFromCoordinate:toCoordinate:

Calculates the distance between two coordinates.

+ (double)getDistanceFromCoordinate:(IBMPosition *)coordinate1 toCoordinate:(IBMPosition *)coordinate2

Parameters

coordinate1

The first coordinate.

coordinate2

The second coordinate.

Return Value

The distance in meters between the two coordinates.

Discussion

The distance between two coordinates is calculated. The result is returned in meters, using a spherical model of the Earth.

isCoordinate:insideArea:

Returns a boolean value based on whether a coordinate lies within an area. Equivalent to calling isCoordinate:insideArea:withBufferZoneWidth:withConfidenceLevel: with a bufferZoneWidth of 0 and a confidenceLevel of IBMConfidenceLevel#LOW.

+ (BOOL)isCoordinate:(IBMPosition *)coordinate insideArea:(id<IBMArea>)area

Parameters

coordinate

The starting coordinate

area

The idIBMArea object to test the coordinate against

Return Value

(BOOL) Returns YES if the coordinate lies inside the area, at the given level of confidence. The dimensions of the area used in this check incorporate any changes specified for the bufferZoneWidth parameter.
