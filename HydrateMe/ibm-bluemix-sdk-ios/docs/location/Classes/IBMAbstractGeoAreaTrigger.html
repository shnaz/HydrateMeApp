bufferZoneWidth

The trigger’s buffer zone width. The value indicates in meters how much to change the area. It can have either a positive or negative value. If it has a positive value, the area becomes bigger. If it has a negative value, the area becomes smaller. All geofence triggers operate on this new area. The default value is 0, which leaves the area unchanged.
