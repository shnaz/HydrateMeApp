Overview

The IBMCloudCode class is a provisional API. APIs that are marked provisional are evolving and might change or be removed in future releases.

IBMCloudCode provides the ability to call cloud code functions defined in the mobile backend

Properties

baseUrl

A property that allows targeting a local Node installation while still leveraging other Mobile Cloud Services (e.g. IBM Data Service or IBM Push). This makes testing Cloud Code easier as it can be done while targeting a local Node server.

@property (nonatomic, copy) NSString *baseUrl
